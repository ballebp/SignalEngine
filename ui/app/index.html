Getting Started

Setup Guide

  1. Open your TradingView chart and add the indicator (AD1, H9S or B5S).
  2. In the indicator settings, set Webhook Key to match the key in Bot Config on the Chart page.
  3. In TradingView Alerts, set the webhook URL to your TradeRelay bot URL.
  4. On the Chart page, expand Bot Configuration, paste your TradeRelay Bot URL and Webhook Code.
  5. Switch to Live mode and enable Auto-Signal on the bot you want to activate.
